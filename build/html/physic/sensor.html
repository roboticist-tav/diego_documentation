Sensor¶

A sensor is an child object of a thing (and/or a child object of a module) that records specific data from its environment. Data is usually communicated through stats and metrics.

Syntax¶

The default declaration syntax is to provide at least the sensor name and type:

       add_sensor({type},moniker);
       add_sensor(moniker)_type(type);

Wheres a sensor is combined with other sensors, this is known as a module. A module is optional, however, more granuality is provided if used. The syntax, is such:

       add_module({type},moniker);
       add_module(moniker)_type(type);
       add_module({type},moniker)_sensor({type},moniker);
       add_module(moniker)_type(type)_sensor(moniker)_type(type);
