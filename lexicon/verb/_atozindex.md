# A to Z Verb Index

## A

| verb | notes<br />`example` | API<br />_see also_ |
| --------- | ----- | ----- |
| `add_` | Initialises creation of proceeding object.<br />`add_camera(front_left_cam)_robot(droid1);` | [add](add.md) |
| `alert_` | Sends an alert message through the appropriate channel to the proceeding object.<br />`alert_human(Harry)_msg(my battery is on fire!);` | [alert](/alert.md) |
| `ask_` | Requests information, to be returned with `tell` verb or postposit.<br />`ask_drone(drone1)_stat(avg_altitude);`<br />`tell_drone(drone1)_stat(avg_altitude)_value(22.893);` | [ask](/ask.md)<br />_see also:_ [tell](/end/md) |

| `do_` | The objectiser verb, parses a verb into an object.<br />`add_go(go_along_blue_route)_robot(droid1)`<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`_route(blue_route)_start(waypoint1);`<br />`do_go(go_along_blue_route);`| |
